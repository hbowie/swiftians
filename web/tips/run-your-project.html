Run Your Project

Tags: Xcode

Type: Xcode Action

When you are ready to build and run your project, use any of the following actions within Xcode:

  • Click on the button with the Play Triangle
  • Use the menu item Product -> Run
  • Use the keyboard shortcut CMD-R
